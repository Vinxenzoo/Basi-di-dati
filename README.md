# 🗄️ Basi di Dati - Gestione Aziendale Universitaria

> **Progetto Universitario - Object Orientation / Database**
> 
> Progettazione e implementazione della base di dati per un sistema di gestione aziendale in ambito universitario.

## 📋 Descrizione del Progetto

Questo repository contiene l'intero ciclo di vita della progettazione di un database relazionale. Il lavoro parte dall'analisi dei requisiti e dalla modellazione concettuale, passando per la ristrutturazione e il diagramma UML, fino ad arrivare alla stesura dello script SQL per la creazione effettiva delle tabelle.

### 🗂️ Contenuto del Repository

All'interno di questo progetto troverai i seguenti artefatti:

*   📄 **`progetto agrillo capasso.sql`**: Lo script SQL completo. Contiene il codice DDL per la creazione dello schema del database e il DML per l'eventuale popolamento dei dati.
*   📊 **`progetto_ristrutturato_uml.mdj`**: Modello UML del sistema strutturato, visualizzabile ed editabile tramite il software **StarUML**.
*   🗺️ **`ristrutturazione capasso-agrillo.drawio`**: Diagrammi e schemi di progettazione, comodamente consultabili e modificabili tramite **Draw.io**.
*   🖼️ **`modifiche apportate non ristrutturato.jpg`**: Un'immagine schematica di supporto che illustra le variazioni e i ragionamenti logici applicati rispetto alla struttura di partenza.
